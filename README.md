## Installation

1. **Clone the repository:**

   git clone https://github.com/jamesprajapati/BargainHunterJS.git
   cd BargainHunterJS

2. **Install dependencies:** Make sure you have Node.js installed. Then run:   
    ```bash
    npm install
    ```
## Running Tests
3. **Run all tests:**
   ```bash
   npx playwright test
  ```
